Arvore de Pasta:
src/ → Contém o código-fonte principal do projeto.

assets/ → Armazena arquivos estáticos (imagens, fontes, ícones, etc.).

config/ → Armazena configurações da aplicação, como banco de dados e variáveis de ambiente.

public/ → Guarda arquivos públicos acessíveis pelo navegador, como index.html.

services/ → Gerencia integrações com APIs e serviços externos.

utils/ ou helpers/ → Contém funções auxiliares e utilitárias.

views/ → Armazena os arquivos de interface do usuário (HTML, templates).
--------------------------------------------------------------------------------------------
models/ → Define a estrutura dos dados no banco de dados.

controllers/ → Contém a lógica de negócio do projeto.

routes/ → Define as rotas da aplicação.
